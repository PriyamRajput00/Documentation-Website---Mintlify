# Mintlify-Inspired Documentation Website (HTML & CSS)

This project is a **desktop-first recreation of a documentation-style website inspired by Mintlify**, built using **only HTML and CSS**.

The focus of this assignment is on **content structure, readability, spacing, typography, and layout accuracy**, closely matching the look and feel of a modern documentation platform.
